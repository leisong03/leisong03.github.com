## `ES6`

### `for-of`

遍历数组：

1.1 `usual` `for`

```js

	for (var i = 0; i < array.length; i++) {   
		console.log(array[i]);   
	}  

```
	
2.2 `ES5` `forEach`

```js

	array.forEach(function (value) {   
		console.log(value);  
	});  

```

问题：不能用break语句跳出循环且不能在这个封闭的函数内使用return语句。

3.3 `for-in`

```js

	for (var i in array) {   
		console.log(array[i]);   
	}  

```

* 数组索引的`i`为`String`: "1","2"...

* 不仅会遍历数组的元素,甚至连expando属性也会遍历出来了

* 遍历数组元素时顺序是任意的

4.4  `ES6` `for-of`

```js

		for (var value of myArray) {   
			console.log(value);   
		}  

```

* 这是目前遍历数组最简洁和直接的语法.

* 它避免了for-in的所有缺陷.

* 它支持break，continue和return。
