## CRC16

每条命令后面都有CRC格式的计算，这里用的算法是ModBus的算法

```c++
uint16_t crc16(uint8_t array[],uint8_t len){
    uint16_t retVal = 0xFFFF;
    for(int i=0;i<len;i++){
	uint16_t temp = array[i];
	retVal ^= (temp&0x00FF);
	for(int j=0;j<8;j++){
	    if(retVal&0x0001){
		//for LSB == 1
		retVal = retVal>>1;
		retVal ^= 0xA001;
	    }else{
		//for LSB == 0
		retVal = retVal>>1;
	    }
	    //retVal &= ~0x8000;
	}
    }
    return retVal;
}
```
